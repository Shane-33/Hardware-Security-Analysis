Rowhammer Tester

A platform for testing the DRAM "Row Hammer" vulnerability

What is Rowhammer?

Rowhammer is a security vulnerability in DRAM (dynamic random-access memory) that allows attackers to manipulate memory cells by repeatedly accessing rows of memory.

How Does Rowhammer Work?

Rowhammer works by rapidly accessing rows of memory in order to cause bit flips in adjacent memory cells. This can lead to memory corruption and potentially enable privilege escalation attacks.

Rowhammer Tester Web Interface
